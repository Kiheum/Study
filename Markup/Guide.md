### 1. 기본 규칙
규칙을 정하여 코드의 통일성을 줌으로써 협업 시 용이할 수 있습니다.

#### 1.1 HTML 문자 인코딩 선언
HTML 문자 인코딩 선언은 meta 태그를 사용하여 기본적으로 utf-8로 설정한다.
``` html
<meta charset="utf-8" />
```

### 2. DTD (Document Type Definition) 선언
DOCTYPE 선언은 HTML5 문서를 기본으로 합니다.

#### 1.1 
``` html
<!DOCTYPE html>
```
